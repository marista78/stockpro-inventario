# MEJORAS MÁS IMPORTANTES 🔥

# 1. FILTROS GLOBALES (MUY IMPORTANTE)

Esto es casi obligatorio.

Arriba debería existir:

- rango fechas
- categoría
- proveedor
- estado
- movimiento
- marca
- unidad

Ejemplo:

```txt
[ Enero - Marzo ▼ ]
[ Electrónica ▼ ]
[ Bosch ▼ ]
[ Bajo stock ▼ ]
```

Esto hace que TODO el dashboard cambie dinámicamente.
